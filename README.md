# Makers60 - Frontend

الموقع الرسمي لإشارات الفوركس والذهب

## www.makers60.com

منصة احترافية لتقديم إشارات التداول والتحليلات الفنية.

### المميزات:
- إشارات الفوركس المباشرة
- تحليلات الذهب والمعادن  
- الرزنامة الاقتصادية
- نظام الاشتراكات المدفوعة
- دعم عربي كامل

### التقنيات:
- React.js + Tailwind CSS
- FastAPI Backend
- MongoDB Database
